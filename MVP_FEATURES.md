# MVP Features

> Version: 1.0
>
> Status: Draft
>
> Purpose:
>
> This document defines the product scope of LVN Atlas.
> Rather than organizing development around technical features, the roadmap is organized around user outcomes.
> Every milestone should solve a meaningful problem before introducing additional complexity.

---

# Product Objective

The primary goal of LVN Atlas is to help people determine whether relocating to another city is financially sustainable before they make one of the biggest decisions of their lives.

Every feature included in the MVP must contribute directly toward achieving this objective.

---

# Milestone 1 — Can the user decide whether relocating is financially possible?

This milestone delivers the minimum experience required for a user to make an informed relocation decision.

## User Capabilities

- Create an account
- Complete a migration profile
- Create a Migration Blueprint
- Generate an estimated monthly budget
- View the Atlas Score™
- Save multiple migration blueprints
- Compare multiple relocation options

---

## Business Value

At the end of this milestone, a user should confidently answer:

> "Can I realistically afford to relocate?"

If the platform cannot answer this question, the MVP is incomplete.

---

# Milestone 2 — Can the platform explain its recommendations?

Generating numbers isn't enough.

Users need to understand *why* those numbers were generated.

## User Capabilities

- View similar migration journeys
- Compare relocation costs between cities
- Understand budget calculations
- Clone an existing migration blueprint
- Customize generated recommendations

---

## Business Value

The platform should become trustworthy rather than acting as a black box.

Users should understand where every recommendation comes from.

---

# Milestone 3 — Can the platform learn from real migration journeys?

Once users begin contributing data, LVN Atlas becomes more intelligent over time.

## User Capabilities

- Submit anonymous migration journeys
- Share monthly expenses anonymously
- Update existing migration data
- Remove shared contributions

---

## Business Value

The platform transitions from using simulated data to continuously improving through community contributions.

Every migration journey helps future migrants make better financial decisions.

---

# Milestone 4 — Can the platform help users plan beyond today?

Relocation isn't just a monthly budget.

It is a long-term financial transition.

## User Capabilities

- View relocation timeline
- Estimate one-time relocation costs
- Compare different salary offers
- Compare housing choices
- Compare transportation strategies
- Forecast long-term savings

---

## Business Value

The platform evolves from expense estimation into relocation decision support.

---

# Milestone 5 — Can the platform scale?

Once the product solves the relocation problem well, attention shifts toward platform maturity.

## Platform Capabilities

- Administration Dashboard
- City Management
- Expense Category Management
- Community Moderation
- Notifications
- Analytics
- Reporting

---

# Explicitly Out of Scope

The following ideas are intentionally excluded from the MVP.

- AI Chat Assistant
- Job Marketplace
- Apartment Marketplace
- Loan Planning
- Insurance Recommendations
- Investment Planning
- Messaging
- Community Feed
- Employer Portal
- International Relocation

These ideas may be explored after the core relocation planning experience has matured.

---

# MVP Success Criteria

The MVP is considered successful when a new user can:

- Create an account
- Complete a migration profile
- Generate a Migration Blueprint
- Receive an Atlas Score™
- Understand the reasoning behind the recommendation
- Save multiple relocation scenarios
- Confidently decide whether relocating is financially sustainable

---

# Product Philosophy

LVN Atlas is **not** an expense tracker.

It is **not** a budgeting application.

It is **not** a cost-of-living calculator.

LVN Atlas is a **Relocation Decision Platform**.

The platform exists to reduce uncertainty by transforming migration experiences into actionable financial planning.
