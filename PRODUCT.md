# Product Requirements Document (PRD)

> Version: 0.1
>
> Status: Draft
>
> Product: LVN Atlas

---

# 1. Executive Summary

LVN Atlas is a migration planning platform designed to help individuals relocate with financial confidence.

Instead of relying on assumptions or scattered advice, users receive personalized migration plans built using data from similar migration journeys, cost-of-living insights, and customizable monthly budgets.

The platform aims to reduce uncertainty by helping users understand the real financial impact of moving before making life-changing decisions.

---

# 2. Problem Statement

Relocating to a new city is financially uncertain.

People often know their future salary but have very little understanding of their future expenses.

Questions such as:

- Can I afford rent?
- How much will groceries cost?
- Can I bring my family?
- How much can I realistically save?
- Which transportation option suits my budget?

are usually answered through random advice, social media posts, or personal assumptions.

Poor financial planning can lead to debt, delayed relocation, stress, or even returning home.

LVN Atlas exists to reduce that uncertainty.

---

# 3. Mission

Help people relocate with confidence by transforming real migration experiences into practical financial planning.

---

# 4. Vision

Become the most trusted relocation intelligence platform where every migration journey helps the next person make better financial decisions.

---

# 5. Target Audience

Primary Users

- Students relocating for education
- Fresh graduates relocating for jobs
- Working professionals
- Families relocating
- Blue-collar workers
- Skilled laborers

Future Users

- International migrants
- Digital nomads
- Remote workers
- Expats

---

# 6. Core Value Proposition

LVN Atlas does not simply track expenses.

It predicts relocation costs by learning from similar migration journeys and helping users build realistic financial plans before they move.

---

# 7. Product Goals

- Reduce relocation uncertainty
- Improve financial planning
- Encourage informed decision making
- Build an anonymous migration knowledge base
- Help future migrants learn from previous journeys

---

# 8. Success Metrics (Initial)

- User completes migration profile
- User generates migration plan
- User customizes budget
- User saves migration plan
- User contributes anonymous expense data

---

# 9. Out of Scope (MVP)

The first version will NOT include:

- Social networking
- Messaging
- Apartment booking
- Loan comparison
- Insurance
- Investments
- AI Chat Assistant
- Job Portal
- Marketplace

These may be explored in future versions.

---

# 10. Current Status

Project is currently in the Product Discovery phase.

No backend implementation has started.

Current focus:

- Product definition
- User research
- Database design
- System architecture
