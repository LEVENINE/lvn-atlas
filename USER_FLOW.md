# User Flow

> Version: 1.0
>
> Status: Draft
>
> Purpose:
>
> This document describes how users interact with LVN Atlas from the moment they discover the platform until they successfully create a Migration Blueprint.
>
> The user flow serves as the foundation for future UI design, API planning, database design, and backend services.

---

# Primary User Flow

```
Landing Page
        │
        ▼
Create Account / Login
        │
        ▼
Complete Profile
        │
        ▼
Create Migration Blueprint
        │
        ▼
Enter Relocation Details
        │
        ▼
Generate Migration Blueprint
        │
        ▼
View Atlas Score™
        │
        ▼
Review Estimated Budget
        │
        ▼
Compare Similar Migration Journeys
        │
        ▼
Customize Budget
        │
        ▼
Save Blueprint
        │
        ▼
Dashboard
```

---

# Step 1 — Discover LVN Atlas

The user learns about LVN Atlas through:

- Search Engine
- Social Media
- Friend Recommendation
- Career Communities
- College
- Employer

### Goal

Understand what the platform does.

---

# Step 2 — Create Account

The user registers using:

- Email
- Password

Future support:

- Google
- GitHub

### Goal

Create a secure personal workspace.

---

# Step 3 — Complete Profile

The user provides information that personalizes recommendations.

Examples:

- Occupation
- Current City
- Monthly Salary
- Family Size
- Lifestyle
- Diet
- Transportation Preference
- Housing Preference

### Goal

Collect enough information to generate meaningful recommendations.

---

# Step 4 — Create Migration Blueprint

The user starts planning a relocation.

Information includes:

- Current City
- Destination City
- Expected Salary
- Planned Relocation Date

### Goal

Create the foundation for financial analysis.

---

# Step 5 — Generate Financial Analysis

LVN Atlas estimates:

- Expected Rent
- Utilities
- Groceries
- Transportation
- Lifestyle Costs
- Emergency Fund
- Expected Savings

### Goal

Build a realistic monthly financial projection.

---

# Step 6 — Calculate Atlas Score™

The platform evaluates relocation readiness.

Factors may include:

- Income
- Savings Potential
- Housing Cost Ratio
- Family Size
- Living Expenses
- Community Data

### Goal

Provide an easy-to-understand relocation confidence score.

---

# Step 7 — Compare Migration Journeys

The platform recommends similar migration experiences.

Users can:

- View anonymous journeys
- Compare budgets
- Learn from similar people
- Understand recommendation reasoning

### Goal

Reduce uncertainty through real-world experiences.

---

# Step 8 — Customize Blueprint

Users modify generated estimates.

Examples:

- Rent
- Groceries
- Transportation
- Utilities
- Personal Expenses

The platform recalculates:

- Budget
- Savings
- Atlas Score™

### Goal

Create a personalized migration plan.

---

# Step 9 — Save Blueprint

Users save multiple blueprints.

Examples:

- Jaipur → Mumbai
- Jaipur → Pune
- Jaipur → Bangalore

### Goal

Compare multiple relocation opportunities.

---

# Step 10 — Dashboard

The dashboard becomes the user's planning workspace.

Users can:

- View saved blueprints
- Compare relocation options
- Edit plans
- Create new blueprints
- Track recommendations

---

# Future User Flows

The following flows are intentionally excluded from MVP.

- Anonymous contribution flow
- Recommendation improvement flow
- Admin moderation
- Notifications
- Community insights
- Timeline simulator
- Mobile onboarding

These will be introduced in future milestones.
